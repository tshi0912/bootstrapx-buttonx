bootstrapx-buttonx
==================

Button extention which based on the button plugin of bootstrap provides the functionality to show the stateful loading icon with loading text.

Usage
---------
set loading state
```javascript
$(this).buttonx('loading');
```

reset to original
```javascript
$(this).buttonx('reset');
```
